1-Crie uma conexão de banco MySQL
2-Crie um arquivo credentials.ini com a seguinte estrutura
[credentials]
host=localhost
user=root
password=sua_senha
database=nome_banco
3-Importe dados "from a single file' do dump mais recente
4-Rode o comando pip install -r requirements.txt no terminal (Python precisa 
  estar no path como variavel de ambient)
5-Execute o arquivo gate_keeper.py
